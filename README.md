# TaskFlow 🚀

Una aplicación full-stack para la gestión de tareas, construida con React, Node.js, PostgreSQL y autenticación JWT.

---

## 🇪🇸 Español

### 📖 Descripción

**TaskFlow** es una aplicación web de gestión de tareas que te permite organizar tu día a día de forma sencilla e intuitiva. Con TaskFlow puedes crear, editar, completar y eliminar tareas, todo en una interfaz limpia y moderna.

Esta aplicación fue construida como proyecto personal para demostrar habilidades en desarrollo full-stack, y está pensada para ser **código abierto**, fácil de usar y extensible.

---

### ✨ Características principales

- ✅ Autenticación de usuarios con **JWT** (registro e inicio de sesión).
- ✅ CRUD completo de tareas (crear, leer, actualizar, eliminar).
- ✅ Diseño responsivo con **Tailwind CSS**.
- ✅ Modo claro y oscuro (próximamente).
- ✅ Fechas de entrega para las tareas (próximamente).
- ✅ Panel de dibujo para acompañar las tareas (próximamente).
- ✅ Código abierto (Licencia MIT).

📖 Description
TaskFlow is a full-stack task management web application that helps you organize your daily tasks in a simple and intuitive way. With TaskFlow, you can create, edit, complete, and delete tasks, all within a clean and modern interface.

This application was built as a personal project to demonstrate full-stack development skills and is designed to be open source, user-friendly, and easily extensible.

✨ Key Features
✅ User authentication with JWT (register and login).

✅ Full CRUD for tasks (create, read, update, delete).

✅ Responsive design with Tailwind CSS.

✅ Dark and light mode (coming soon).

✅ Due dates for tasks (coming soon).

✅ Drawing board to accompany tasks (coming soon).

✅ Open source (MIT License).
