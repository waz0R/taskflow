
# TaskFlow

Aplicación full-stack para la gestión de tareas, construida con React, Node.js, PostgreSQL y autenticación JWT.

---

## 📘 Español

### Descripción

TaskFlow es una aplicación web de gestión de tareas que permite organizar el día a día de forma sencilla e intuitiva. Ofrece un CRUD completo de tareas, autenticación de usuarios y una interfaz moderna y responsiva.

Este proyecto fue creado como parte de un plan de aprendizaje en desarrollo full-stack y está pensado para ser código abierto, fácil de usar y extensible.

### Características actuales

- Autenticación de usuarios con JWT (registro e inicio de sesión).
- CRUD completo de tareas (crear, leer, actualizar, eliminar).
- Diseño responsivo con Tailwind CSS.
- Código abierto bajo licencia MIT.

### Características planificadas (próxima versión)

- Fechas de entrega para las tareas.
- Modo claro y oscuro.
- Subida de imágenes a las tareas.
- Panel de dibujo integrado.
- Filtros por estado y fecha.
- Paginación en la lista de tareas.
- Documentación de la API con Swagger.
- Selector de idioma (español/inglés).

### Tecnologías utilizadas

- Frontend: React, Tailwind CSS, Axios, React Router.
- Backend: Node.js, Express, PostgreSQL, JWT, bcrypt.
- Despliegue: Pendiente (Railway / Vercel).

### Instalación y uso local

Clonar el repositorio:

```bash
git clone https://github.com/waz0R/taskflow.git
cd taskflow
Configurar el backend:

bash
cd backend
npm install
# Crear archivo .env con las variables de entorno
npm start
Configurar el frontend:

bash
cd frontend
npm install
npm start
Abrir el navegador en http://localhost:3000.

Description
TaskFlow is a full-stack task management web application designed to help organize daily tasks in a simple and intuitive way. It offers a complete task CRUD, user authentication, and a modern, responsive interface.

This project was built as part of a full-stack development learning plan and is intended to be open source, easy to use, and extensible.

Current features
User authentication with JWT (register and login).

Full task CRUD (create, read, update, delete).

Responsive design with Tailwind CSS.

Open source under MIT license.

Planned features (next version)
Due dates for tasks.

Dark and light mode.

Image upload for tasks.

Integrated drawing board.

Filters by status and date.

Pagination in the task list.

API documentation with Swagger.

Language selector (Spanish/English).

Technologies used
Frontend: React, Tailwind CSS, Axios, React Router.

Backend: Node.js, Express, PostgreSQL, JWT, bcrypt.

Deployment: Pending (Railway / Vercel).

Local installation and usage
Clone the repository:

bash
git clone https://github.com/waz0R/taskflow.git
cd taskflow
Set up the backend:

bash
cd backend
npm install
# Create a .env file with environment variables
npm start
Set up the frontend:

bash
cd frontend
npm install
npm start
